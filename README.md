# 🖨️ ASCII Art & Yazı Dönüştürücü Sitesi

Düz yazıları anlık olarak siber/hacker tarzı devasa blok grafik yazılara dönüştüren, tek dosyadan (`index.html`) oluşan modern ve hızlı bir web aracıdır.

## 🚀 Özellikler
* ⚡ **Anlık Dönüşüm:** Yazı yazıldığı anda, hiçbir gecikme ve takılma olmadan doğrudan ASCII koduna dönüşür.
* 📋 **Tek Tıkla Kopyala:** Üstteki kopyala butonu sayesinde metni bozmadan direkt kopyalayıp Discord kanalına yapıştırabilirsiniz.
* 🟢 **Cyberpunk Tema:** Tamamen geliştirici stili, göz yormayan neon yeşili ve koyu tema tasarımı.

## 🛠️ Kurulum ve Canlıya Alma
1. Bir GitHub deposu (repository) oluşturun ve aşağıdaki `index.html` dosyasını içine yükleyin.
2. Deponuzun **Settings > Pages** sekmesine girin.
3. Branch ayarını **`main`** (veya master) yapıp **Save** butonuna basın.
4. Birkaç dakika içinde size verilen ücretsiz link üzerinden siteniz dünya çapında yayına girecektir!
